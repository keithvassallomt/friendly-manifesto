# The Friendly Manifesto

A voluntary set of principles for open-source Software Developers, Gatekeepers, and Users.

**Software Developers** commit to building honest, secure, and respectful software that does not exploit its Users.
**Gatekeepers** commit to distributing software fairly, transparently, and without subjective or ideological gatekeeping.
**Users** commit to engaging with developers and communities respectfully, in good faith, and with realistic expectations.

## The Website

The website is built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com). It includes reading mode themes (dark, sepia, and a dyslexia-friendly option) and is designed to meet WCAG 2.1 AA accessibility standards.

### Running locally

```bash
npm install
npm run dev
```

### Building for production

```bash
npm run build
npm run preview
```
