# Changelog

All notable changes to The Friendly Manifesto will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-03-11

### Added

- The Friendly Manifesto v1.0 in English (UK) and English (US)
- Website built with Astro and Tailwind CSS
- Reading mode with four themes: default, dark, sepia, and dyslexia-friendly
- Language switcher with flag icons
- Badge section with copy/download options for adopters
- WCAG 2.1 AA accessibility support (skip links, keyboard navigation, reduced motion, focus indicators)
- Adopter list powered by `src/data/adopters.json`
- Contributing guide with instructions for adoption, translation, and new languages
